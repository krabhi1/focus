Focus enforces your session rules.

One daemon tracks active tasks, starts breaks, applies cooldowns, and relocks the screen when required.

Install Focus Latest release

Features

  • Preset task durations: short, medium, long, deep.
  • Break windows for long and deep tasks with lock enforcement.
  • Cooldown after task completion before next task is allowed.
  • Relock on unlock during break/cooldown based on relock delay.
  • Completed task history persisted to file.
